# Bitcoin Exchange plateform

An utterly fantastic bitcoin exchange plateform done with Django 1.11.
+ Conceptual platform. Not connected to the blockchain.
+ Design by myself.

## Features

- Account Creation
- Authentification
- Send BTC
- Receive BTC
- Add Contacts
- Transaction list
- Error handling
